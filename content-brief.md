# Content Brief — 第一版內容產出指引

**用途**：你在此用條列寫下關鍵資訊與想法 → **實作 Agent** 依此 + [content-schema.md](content-schema.md) 產出 `content/zh/*` 初版（約 60 分）→ **內容 Agent** 從初版迭代完稿。  
**填寫方式**：想到什麼寫什麼，條列即可；不必成段。Agent 會依 schema 長度與結構整理成可上線文案。

未來會長什麼樣？
AI 的時代，人類將徹底從生產函數中的要素抽離（經濟學來看 - land, capital, labor）
勞動經濟學轉變為 agent economy
一個從 input 到 function 到 output 的金字塔將重新被定義
input 是底端的基石，三大要素的稀缺性限制整個人類文明：時間、能源、材料
時間原本是最大瓶頸，因為所有基本生產工作都有人作為勞動的必要投入
未來成為 agent 以後人將大幅從最底層抽離，agent 所需耗用的時間將只剩下物理限制，且可持續被加速
能源是下一個瓶頸，從石化能源轉向太陽能源是擺脫能源稀缺的方向，也在進行中
最後材料將成為瓶頸，因此循環材料的推進是未來重點
當 input 稀缺性打開的過程中，function 層的兩大重點將更為重要 - creation 與 governance
涉及如何創造這整個體系（agent 在其中的運作體系），以及如何治理與分配產出價值
然而最重要的是最上層 output - 人本
以人為本，花更多時間去探究人本身的需求，才能清楚知道我們為何而生產

華爾街到矽谷
資本與軟硬體人才已經全面結合，AI 到 AGI 的發展 agent 全面取代人類成為 labor 勢不可擋
但是已經被規訓成生產機器的多數大眾，該如何重新找回從小學就被剝奪 - 身而為人的價值，仍是未知數
因此這個宣言要宣告的就是如何推動全面人本
我們不只要讓少數統治者人本，更要解放多數被統治者也實踐人本
我們將以科技與設計作為手段
以人為核心，從人力 labor 到人才 human 的過渡，將往左右兩側畫出一個無限循環
一邊是需求端：職場 - 持續從個人到團隊到組織，加速 AI 導入到工作中，解放勞力，重塑職場上的人才需求，改變生產模式
一邊是供給端：從成人回溯孩童 - 改變世界不能寄託於孩童，孩子從對成人世界的模仿學習而非課本（身教大於言教），所以要從成人端開始，協助在職場上仍在探索、找不到自我定位的人找回自我，透過 designing your life 等方法論，幫助人們找回人生自主權，接著協助掌握 AI 時代的能力，讓他們自我學習，每個人都走出專屬自己的一條路，最終走向充滿心流的人生 - Empower people into the flow

這不是傳統線性思維能達到的
這會是一個循環的系統觀，唯有兩邊同時轉動，才能成為一個逐漸上升的飛輪
讓 AI 時代技術推進的同時，人力往人才迭代的速度也能跟上
